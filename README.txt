KATIE BARLOW AUTHOR WEBSITE

FILES
- index.html
- styles.css
- script.js

HOW TO PREVIEW
1. Unzip the folder.
2. Open index.html in a web browser.

IMPORTANT EDITS
1. In script.js, replace YOUR-EMAIL-HERE with your author email address.
2. In index.html, replace the # on the Buy the Book button with your Amazon or Barnes & Noble link.
3. Replace the author photo placeholder with your photograph.
4. Replace the illustrated book-cover placeholder with your real cover image.
5. Add event dates and locations.

PUBLISHING OPTIONS
This static site can be uploaded to most web hosts, including Netlify, GitHub Pages, Cloudflare Pages, or a traditional hosting provider. It can also be used as a visual blueprint for Squarespace or Wix.
