HOW TO REPLACE THE CURRENT WEBSITE
----------------------------------
1. Open:
   https://github.com/kbarlow0126/katiebarlowauthor

2. Click Add file > Upload files.

3. Open this extracted folder and select ALL seven website files listed above.

4. Drag them into GitHub.

5. GitHub will replace index.html, styles.css, script.js, and the existing images.
   That is correct.

6. Commit message:
   Replace site with final Stanley website

7. Click Commit changes.

8. Wait 2–5 minutes.

9. Open:
   https://kbarlow0126.github.io/katiebarlowauthor/

10. Press Ctrl + F5 once to force a fresh version.
